# 1 Hz VO2master app recording from Moxy

Exported from VO2 Master Manager app, recorded at 1 Hz. A short example
recording with three sensors.

## Format

.csv file with 12 columns and 240 rows.

- `nirs_channels = c("SmO2[%]", "SmO2 - 2[%]", "SmO2 - 3[%]", "THb[THb]", "THb - 2[THb]", "THb - 3[THb]")`

- `time_channel = c("Time[s]", "Time[hh:mm:ss]")`
